# Sprint 7 — Projecte Power BI

Aquest repositori conté el projecte desenvolupat amb Power BI durant l’Sprint 7.

## Objectius del projecte
- Analitzar dades de vendes i transaccions.
- Construir dashboards interactius amb Power BI.
- Crear mesures i KPIs amb DAX.
- Aplicar bones pràctiques de visualització de dades.
- Millorar la llegibilitat i la comprensió analítica.

## Dashboards inclosos

### 1. Global Sales Dashboard
- Anàlisi mensual de vendes
- Empreses per país
- Comparació de KPIs
- Visió global de vendes

### 2. Germany Sales Analysis
- Tendència mensual de vendes (2018–2019)
- Anàlisi d’objectius de vendes
- KPI de mitjana de vendes a Alemanya
- Seguiment d’objectius empresarials

### 3. User Transactions Analysis
- Productes comprats per usuari/ària
- Anàlisi de la mitjana de transaccions per usuari/ària
- Producte més car i més barat
- Distribució geogràfica de transaccions

## Tecnologies utilitzades
- Power BI
- DAX
- Modelatge relacional de dades
